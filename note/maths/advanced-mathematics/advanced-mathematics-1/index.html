# 映射与函数

# 映射

设XXX，YYY 是两个非空集合，如果存在一个法则fff，使得对XXX 中每个元素xxx，按法则fff 在YYY 中有唯一确定的元素yyy 与之对应，那么称fff 为从XXX 到YYY 的映射。记作：

f:X→Yf:X\to Yf:X→Y

其中yyy 称为元素xxx（在映射fff) 下的像；xxx 称为元素yyy（在映射fff）下的原像；集合XXX、xxx 所有像的集合分别称为映射fff 的定义域和值域，分别被记作DfD_fDf​ 和RfR_fRf​

其中需要注意的有：

  1. 定义一个映射必须具备定义域DfD_fDf​，值域RfR_fRf​ 和对应法则fff。
  2. 对于每个xxx，像是唯一的；但对于每个yyy，其原像不一定是唯一的。

若YYY 中所有yyy 都是XXX 中某元素的像，则称fff 为XXX 到YYY 上的映射或满射；若对XXX 中任意两个不同元素x1≠x2x_1\neq x_2x1​=x2​，他们的像不相等，则称fff 为XXX 到YYY 的单射；若映射fff 既是单射又是满射，则称fff 为一一映射或双射。

映射又被称为算子。

按上述定义，只有单射存在逆映射。

# 函数

函数定义不作赘述。总之，如果两个函数的定义域和对应法则都相同，那么这两个函数相同。

# 初等函数

幂函数、指数函数、对数函数、三角函数和反三角函数统称为基本初等函数。

由常数和基本初等函数经过有限次的四则运算和有限次的函数复合步骤所构成并可用一个式子表示的函数，称为初等函数。

分段函数是否是初等函数存在争议。

双曲正弦等含指数函数常用双曲函数在 P13。

# 数列的极限

# 定义

数列{an}\{a_n\}{an​}，如果存在常数aaa，对于任意给定的正数ε\varepsilonε（不论它多么小，即不论它多接近0+0^+0+），总存在正整数NNN，使得当n>Nn>Nn>N 时，不等式

∣xn−a∣<ε|x_n-a|<\varepsilon∣xn​−a∣<ε

都成立，那么称常数aaa 是数列{an}\{a_n\}{an​} 的极限，或者称数列{an}\{a_n\}{an​} 收敛于 a。

如果不存在这样的常数aaa，就说数列{an}\{a_n\}{an​} 没有极限，或者说数列{an}\{a_n\}{an​} 是发散的。

# 收敛数列的性质

  1. 极限的唯一性
    如果数列{an}\{a_n\}{an​} 收敛，那么它的极限唯一。
  2. 收敛数列的有界性
    如果数列{an}\{a_n\}{an​} 收敛，那么数列{an}\{a_n\}{an​} 一定有界。
  3. 收敛数列的保号性
    如果lim⁡n→∞an=a\lim\limits_{n\to\infin}a_n=an→∞lim​an​=a，且a>0a>0a>0（或a<0a<0a<0）
    推论：如果数列{an}\{a_n\}{an​} 从某项起有an≥0a_n\geq 0an​≥0（或an≤0a_n\leq 0an​≤0），且lim⁡n→∞an=a\lim\limits_{n\to\infin}a_n=an→∞lim​an​=a，那么a≥0a\geq 0a≥0（或a≤0a\leq 0a≤0）。
  4. 收敛数列与其子数列间的关系
    如果数列{an}\{a_n\}{an​} 收敛于 a，那么它的任一子数列也收敛，且极限也是aaa。

发散数列{an}\{a_n\}{an​} 的子数列也可能是收敛数列。
